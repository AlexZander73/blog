Social Post Helper: Shipping a Useful Non-AI MVP

I wanted a social drafting tool that was useful before any AI showed up. Social Post Helper started as a hard constraint: make the workflow lighter without pretending the app understands images.

March 12, 20262 min read

Apps Tools Launches

Starting from a real gap

I started Social Post Helper because I was tired of the gap between two bad options. On one side was doing everything by hand every time I wanted to turn a few photos into a decent post. On the other side was jumping straight into AI tooling, cloud APIs, and a level of complexity that felt out of proportion to the actual problem. I felt oddly stubborn about that gap. I wanted something simpler and more honest.

A narrow first version on purpose

So the first version stayed narrow on purpose. I built a local-first Next.js app that takes uploaded photos or a single video, extracts metadata, lets me add a little manual context, and then generates deterministic drafts for Instagram, Facebook, and YouTube. The concrete change was not one flashy feature. It was the whole workflow: upload, summarize detected info, generate rule-based drafts, edit them in place, and export them without any model call hiding in the background.

That decision forced a tone on the product. The app can be helpful, but it cannot bluff. If there is GPS metadata, it can use it. If there is a date, it can format it. If I write notes, it can fill templates with them. If none of that exists, the drafts have to stay plain. I actually like that. It keeps the product from sounding smarter than it is.

What the constraint taught me

The biggest lesson from day one was that constraint can be a feature. By refusing AI in the MVP, I had to make the underlying workflow clean enough to stand on its own. That pushed me toward better structure, clearer user inputs, and a modular architecture that can absorb richer generation later instead of depending on it now.

Next move: put the first real users through the workflow and learn where deterministic drafting saves time and where it still feels thin.
